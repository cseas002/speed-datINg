# Speed Dating Event - Participant Data Template

This Excel file should contain the following columns:

| Column | Type | Description | Example |
|--------|------|-------------|---------|
| Name | Text | Participant's full name | John Smith |
| Email | Text | Participant's email address | john@example.com |
| Sex | Text | Participant's sex | male / female |
| Partner sex preference | Text | Preferred sex of dating partner | female / male |
| About Me | Text | Self description and personality traits | I'm outgoing, love hiking and travel. Career-driven professional. |
| Looking For | Text | What they're looking for in a partner | Seeking someone with similar values, enjoys adventures |
| Personality | Text | Personality characteristics (Myers-Briggs, Big Five, etc.) | ENFP, Extroverted, Creative |
| Arrived | Text | Whether participant attended the event | Yes / No |

## Instructions

1. Export your Google Form responses to Excel
2. Add/update the columns to match the format above
3. Fill in the "Arrived" column with "Yes" or "No"
4. Only participants with "Yes" in the Arrived column will be included in matching
5. Upload the file through the Admin Dashboard

## Sample Data

```
Name,Email,Sex,Partner sex preference,About Me,Looking For,Personality,Arrived

Mike Davis,mike@example.com,male,female,Tech enthusiast and coffee lover,Intelligent and kind person,INTP,No
```
